# 영상 컨셉

## 앵글
고려 474년을 한 편의 드라마로 — 건국부터 멸망까지, 왕이 바뀔 때마다 백성의 삶이 어떻게 뒤집혔는지를 인과 사슬로 엮는다. 레퍼런스의 검증된 연대기 서사 구조를 계승하되, "현종 보강 + 조선 대비 심화 + 백성 관점 강화"로 빈틈을 채운다.

## 핵심 약속
이 영상을 끝까지 보면, 고려가 왜 조선과 완전히 다른 나라였는지 소름 돋게 깨닫고, '우리 역사에 이런 나라가 있었어?'라는 탄식이 나온다.

## 타겟 시청자
조선 역사는 드라마로 접했지만 고려는 '고려청자'와 '몽골 침입' 정도만 아는 사람. "고려가 뭐가 다른데?"라는 가벼운 호기심을 가진 20~40대.

## 서사 유형
시간순 서사 (건국→위기→결말) — 465만 조회로 검증된 구조. "그 다음은?" 궁금증 자연 유지, 중반 이탈 위험 낮음.

## 감정 전략
- 목표 감정: 지적 쾌감 (댓글 좋아요 3,500으로 검증)
- 감정 아크: 호기심(건국) → 통쾌함(광종 개혁·서희 담판) → 분노/비장함(무신정권·몽골·원 간섭) → 통쾌함+씁쓸함(공민왕) → 숙연함(멸망)
- 리텐션 접근: 파트 전환점마다 오픈루프 + 중반 패턴 인터럽트로 긴장 재점화

## 제목 후보
1. 15분간 옛이야기 듣듯 정리하는 '고려 474년의 흥망성쇠'
2. "조선과 딴판이었습니다" 알수록 소름 돋는 고려 500년의 비밀
3. 고려는 왜 조선과 완전히 달랐을까 — 500년 통째로 정리

## 확정 제목
"조선과 딴판이었습니다" 알수록 소름 돋는 고려 500년의 비밀

## 썸네일 텍스트 후보
1. 15분 정리 + 조선이 숨긴 + 고려 500년 (5자 + 5자 + 7자)
2. 15분 몰입 + 조선과 딴판이었던 + 진짜 고려 (5자 + 9자 + 5자)
3. 500년 총정리 + 우리가 몰랐던 + 고려의 반전 (7자 + 7자 + 6자)

## 썸네일 이미지 컨셉

### 컨셉 A: 왕조의 웅장함 (스케일 대비)
- 구도 패턴: 스케일 대비
- 핵심 오브젝트: 고려 궁궐(만월대) + 하단 백성 실루엣 — 상단 2/3
- 색채/대비: 어두운 갈색·먹색 배경 + 주황색 포인트 텍스트
- 감정 톤: 웅장함 + 호기심

### 컨셉 B: 시대 전환 (Before→After)
- 구도 패턴: Before→After (좌: 고려 전성기 / 우: 몽골 침입기)
- 핵심 오브젝트: 좌: 벽란도 무역항 활기찬 장면 / 우: 불타는 개경 — 상단 2/3
- 색채/대비: 좌: 따뜻한 금빛 / 우: 차가운 회청색. 중앙 분리선
- 감정 톤: 격차 충격 + 비장함

## 텐션 트라이앵글 시너지
- 제목: "소름 돋는 고려 500년의 비밀" → 정보적 약속 + 감정 보상 예고
- 썸네일 텍스트: "조선과 딴판이었던 / 진짜 고려" → 비교 욕구 + 감정적 호기심
- 썸네일 이미지: 웅장한 궁궐 + 백성 → 시각적 규모감 + 서사 기대
- 시너지 평가: "조선과 뭐가 다른데?" + "소름 돋는 비밀" + 웅장한 비주얼 = 클릭 충동

## 선택 근거
레퍼런스 465만 조회의 연대기 서사 구조를 충실히 계승. 조선 대비 심화 + 현종 보강 + 백성 관점으로 빈틈을 채워 차별화. 종합 평가 최상위.
