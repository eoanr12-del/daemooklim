# 대본 검수 결과

## 핵심약속 검수

- [x] Hook에서 핵심약속을 바로 이행하고 있는가 — "한 남자의 복수가 대륙을 불태웠습니다"로 복수 서사를 즉시 선언하고, "4천만 명이 목숨을 잃었거든요"로 백성 관점까지 Intro에서 제시. 미루지 않음.
- [x] 각 파트가 핵심약속 이행에 기여하는가 — 파트1(복수 동기) → 파트2(전쟁 기계 탄생, 유목민 부품화) → 파트3(금나라 정복+농민 피해) → 파트4(호라즘 학살+백성 묘사) → 클로징(유산과 반전). 모든 파트가 "복수 서사 + 백성 관점" 핵심약속에 기여.
- [x] 마무리에서 핵심약속이 완결되는가 — "4천만의 피 위에 세워진 평화. 정복의 칼날 아래 죽어간 사람들의 이름은 역사에 남지 않았습니다"로 핵심약속 완결.

## 구조 검수

- [x] outline.md의 본문 구조(파트 순서, 내용)대로 작성되었는가 — Hook/Intro, 파트1~4, 클로징 순서 및 내용 모두 outline.md와 일치.
- [x] 확정 제목이 암시하는 구조를 정확히 따르는가 — "복수 → 정복 → 숨겨진 진짜 이야기" 구조를 시간순으로 충실히 따름.
- [ ] 타겟 러닝타임 범위 안에 들어가는가 — **타겟 6,000자(15분), 실제 약 4,349자(공백 제외). 목표 대비 약 72%로 상당히 부족.** 특히 파트1(733자, 목표 1,000자의 73%), 클로징(578자, 목표 800자의 72%)이 가장 짧음.
- [x] Hook과 Intro가 outline.md와 일치하는가 — outline.md의 Hook/Intro 텍스트와 정확히 일치.
- [x] 각 파트의 감정 흐름이 outline.md 설계와 맞는가 — 공감+연민(파트1) → 경외감+서늘함(파트2) → 통쾌함+공포(파트3) → 공포+먹먹함(파트4) → 여운(클로징). 설계와 부합.
- [x] 리텐션 장치(오픈루프, 전환 멘트)가 파트 전환 지점에 배치되었는가 — 모든 파트 전환에 오픈루프/전환 멘트 배치 확인. 퀴즈형 장치(파트3 인형 말)도 포함.

## verdict: 수정

### 수정 사유
치명적 이슈는 없으나, **중요 이슈 3건**이 존재:
1. 분량이 타겟 대비 약 72%로 상당히 부족하여 15분 영상 기준에 미달
2. 역참 1,400여 개의 범위를 "대륙 전체"로 과장 (사실 오류에 준하는 부정확)
3. 영토 2배 확장을 오고타이 단독 업적으로 오귀속

분량 부족이 가장 큰 문제로, outline.md에 설계된 세부 내용(백성 관점 장면, 에피소드 디테일)을 추가하여 목표 분량에 근접시켜야 함. 역참 범위와 영토 확장 주체는 간단한 문구 수정으로 해결 가능.
