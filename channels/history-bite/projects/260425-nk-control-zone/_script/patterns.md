# 패턴 추출 — 북한 정치범 수용소 (단일 레퍼런스 N=1)

> ⚠️ 레퍼런스 1개(역썰남, 17:40, 조회수 9.7만, 좋아요 1,412, 구독 9,940). 신생 채널이지만 조회수/구독비 매우 높음(약 9.8배) — 이 한 영상의 패턴을 채널 프로필(history-bite)과 결합해 우리 영상 설계 원칙으로 변환.

---

## 1. 반복되는 훅 공식

**관찰**:
- 레퍼런스 훅: 거대 통계가 아니라 **"한 사람의 얼굴"**부터. "엄마 두부 한 접시"라는 가장 작은 디테일 → 가장 큰 충격(우크라 포로)으로 점프 → "죽음보다 무서운 게 뭐?" 미스터리 박기.
- 인트로 패턴: 즉시 "3대 연좌"의 충격 + 교과서 악명(나치/굴라크/우간다)와의 비교 → "그런데 더한 게 지금 한반도에"로 시간 비틀기.

**설계 원칙**:
1. **첫 5초에 통계 말고 한 사람**: "67만 명이 갇혀 있어요"가 아니라 "9살에 트럭에 실린 한 아이가 있었습니다".
2. **그 한 사람의 가장 작은 디테일 1개**(두부 한 접시, 구슬치기 하다 끌려간 아이) — 공감 무기.
3. **인트로에서 약속 명시**: "끝까지 보면 [그 한 사람이 어떻게 됐는지 + 시스템의 정체] 둘 다 알게 됩니다".
4. history-bite 톤("~거든요/~잖아요") + 금지 표현("학자들에 의하면", "~했다고 합니다") 회피.

## 2. 반복되는 본론 구조 & 전달 기법

**관찰**:

*구조*: **현재 한 사람(2026) → 과거 한 가족(1977) → 시스템 설계(1958) → 일과 디테일(24시간) → 사라지는 사람들(여성/아이) → 다시 현재 한 사람** — **수미상관 + 폭로 서사 + 시간 점프**.

*전달 기법*:
- **비유**: "학교로 치면 반장, 관심 대상, 퇴학 명단", "조선의 연좌제 + 중국의 하방 + 소련의 굴라크 = 한 냄비에 끓였더니" (추상 → 일상)
- **숫자**: 6.5만/19만/549km²(서울)/A4 7장/옥수수 25g/한국 노래 70곡 — 작은 단위(7장, 25g)와 거대 단위(549km², 19만)의 교차로 스케일 흔들기
- **사례**: 1977 강철환(9살) / 2022 22살 청년 공개처형 / 2026 우크라 포로 청년 — 이름·연도·나이 구체

*리텐션 장치*:
- 미스터리 감추기: "왜 못 나왔는지는 뒤에서 말씀드릴게요. 일단 기억만 해 두세요."
- 대비 충격: "20살 건장한 청년이 가장 먼저 쓰러집니다"
- 같은 민족 충격: "아우슈비츠는 다른 민족한테 한 짓이었습니다. 여기선 같은 민족이..."
- 수미상관 클로징: 두부 한 접시 → 두부 한 접시 기억하는 사람이 한 명도 남지 않는다

**설계 원칙**:
1. **본론은 수미상관 폭로 서사**: 한 사람으로 시작 → 시스템 설명 → 그 한 사람으로 닫기.
2. **시간 점프 의무**: 2026 → 1977 → 1958 → 다시 2026. 시간 띠를 흔들어 "지금 살아있는 문제" 감각.
3. **"앞에서 잠깐 나왔던 그 사람" 회수**: 강철환 할아버지·삼촌이 못 나온 이유를 본문 후반에 회수.
4. **숫자는 1~2분에 1번**, 작은 수치(A4 7장, 25g)와 거대 수치(549km², 19만)를 교차.
5. **history-bite 시그니처 의무 삽입**: "그런데 여기서 소름끼치는 게 있습니다", "자, 이게 진짜 반전입니다" 각 1~2회.

## 3. 감정 증폭 전략

**관찰**:

*댓글 감정 분석*:
1. **공감·감사** (자유의 가치, 운, 6.25 유공자 — 다수)
2. **분노·명명** ("김일성이 지옥으로", 202좋아요)
3. **교육 욕구** ("초등학교에서 가르쳐야", 145좋아요)

*감정을 만든 장치 역추적*:
- 공감: 두부 한 접시, 9살 강철환 구슬치기
- 분노: "한 마디였을 뿐인데", "같은 민족이 동포 아이한테"
- 무력감: "두부 한 접시 기억하는 사람이 한 명도 남지 않는다"

*감정 흐름*: **공감(훅) → 분노(연좌제·1958 분류표) → 경악(상납·낙태) → 위기감(2026 위성·공개처형) → 진중한 결심(클로징)**

**설계 원칙**:
1. **분노는 깊게, 짧게**: 공개처형 사례는 1번만. 디테일 깊지만 길게 끌지 않음.
2. **클로징은 "기억" 톤**: 분노로 끝내지 말 것. "두부 한 접시", "그때 백성", "기억하는 사람이 있어야"로 진중하게.
3. **백성 관점 일관**: "그때 백성들은 어땠을까?"를 "지금 그곳 사람들은 어떨까?"로 자연스럽게 변주.
4. **댓글의 "운/감사" 코드 흡수**: "우리가 이쪽에 태어난 게 그저 운이었다는 생각" 같은 자각 멘트 클로징에.
