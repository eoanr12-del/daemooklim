## 패키지 A: "돈의 방향"

### 컨셉
- 앵글: **역발상 논증** -- "물량 넘기기"라는 시청자 1순위 의심을 정면으로 다루면서, "말이 아니라 돈을 따라가라"는 판단 프레임을 제시. 월가가 입으로는 뭐라 하든 실제 자금 집행 데이터(블랙록 비중확대, JP모건 29조 투자, 모건스탠리 위험자산 확대)가 향하는 방향이 진짜라는 논증.
- 핵심 약속: "월가가 매수를 외치면 물량 넘기기라고 생각하셨죠? 이 영상 끝까지 보시면, 말이 아니라 돈의 흐름으로 시장 방향을 읽는 판단 기준을 갖게 됩니다"
- 타겟 시청자: 최근 상승장에 올라타지 못하고 관망 중인 직장인 투자자. "월가가 사라는데 또 물리는 거 아니야?"라는 불안 속에서 근거 있는 판단 기준을 원하는 사람.
- 서사 유형: **역발상 논증** -- 통념("월가 매수 권고 = 물량 넘기기") 반박 → 진짜 근거 제시. 댓글 상위 4개가 모두 불신/회의이므로 이를 정면 돌파하는 앵글이 가장 강력. 레퍼런스 001과 가장 유사한 검증된 방향.
- 감정 전략: **통쾌함(인사이트 획득)** / 불안·의심(초반 20%) → 데이터 기반 "아하" 전환(중반 40%) → FOMO(중후반 20%) → 통쾌함+판단 자신감(후반 20%) / 리텐션 접근: "물량 넘기기"를 초반 2분 내 직접 언급하여 이탈 방지 + 데이터가 하나씩 쌓이며 통념이 무너지는 쾌감으로 중반 견인

### CTR

**제목 후보:**
1. `"물량 넘기기 아닙니다" 블랙록이 지금 '매수'로 돌아선 진짜 '이유'` (52자, 따옴표+역발상+비밀/진실 패턴, 레퍼 001 구조 L1 차용: "현금 미리 준비하세요" → "물량 넘기기 아닙니다" 교체, 댓글 1순위 의심을 정면 해소하여 호기심 갭 극대화)
2. `월가가 전쟁 D-7에 '매수' 버튼 누른 이유, 블랙록·JP모건·모건스탠리 전부` (42자, 자유형식, 숫자리스트+스토리훅 결합, 3개 기관 나열로 권위 집중 + "전부"로 스케일감)
3. `"말 말고 돈을 봐라" 블랙록 29조 베팅이 알려주는 다음 주 시장 방향` (41자, 따옴표+결과/증거 패턴, 29조라는 구체 수치+시급성 조합)

**확정 제목:** 후보 1 -- `"물량 넘기기 아닙니다" 블랙록이 지금 '매수'로 돌아선 진짜 '이유'`
- 근거: 레퍼 001 제목 구조(따옴표 인용+권위 주체+진짜 이유)를 L1 차용하면서, 댓글 최다 반응인 "물량 넘기기" 의심을 따옴표 문구에 직접 배치. 시청자의 기존 의심을 제목에서 먼저 건드려 호기심 갭이 가장 강함. 52자로 모바일 잘림 기준 충족.

**썸네일 텍스트:**
① `휴전 종료 D-7` + `"역대급 매수"` (8자 + 7자 = 15자, 2줄 메인) -- L1 구조 차용: 레퍼 001 "전쟁 휴전 종료까지 D-7" + "정말 무섭게 오릅겁니다"의 [위기+역설적 결과] 구조 그대로, 키워드를 "역대급 매수"로 교체. 따옴표로 인용감 부여.
② `물량 넘기기?` + `"29조 베팅"` (6자 + 6자 = 12자, 2줄 메인) -- L2 키워드 차용: 댓글 핵심 키워드 "물량 넘기기"를 의문형으로 던지고, 29조 수치로 반박하는 구조. 레퍼 001의 "구어체 긴급감" 톤 차용.
③ `"전쟁인데 왜 사?"` + `돈은 알고 있다` (9자 + 7자 = 16자, 2줄 메인) -- L3 멀티 리믹스: 레퍼 001 의문형 자기투영("왜 나만 못 들어가지?") + 001 인사이트 멘트("다이먼의 말이 아니라 돈이야") 조합. 시청자 의문을 대변하면서 답의 단서 제공.

**이미지 컨셉:**
A. **스마트머니 자금 흐름** (화살표/그래프 + 인물 감정)
- 구도 패턴: 화살표/그래프 + 인물 감정
- 핵심 오브젝트: 거대한 녹색/빨간 상승 화살표 + 정장 차림 월가 트레이더(결의에 찬 표정)
- 색채/대비: 배경 짙은 남색/검정 + 포인트 빨강·녹색 화살표 + 노란 텍스트
- 감정 톤: 긴장감 속 결의(tension + determination)

B. **전쟁 vs 달러** (A vs B 대결)
- 구도 패턴: A vs B 좌우 분할
- 핵심 오브젝트: 좌측 폭발/연기(전쟁 이미지) vs 우측 달러 다발/상승 차트
- 색채/대비: 좌측 어두운 적색/주황(전쟁) vs 우측 짙은 녹색/금색(돈). 중앙 경계선에서 색채 충돌.
- 감정 톤: 충격적 대비(인지 충돌 -- 전쟁인데 돈이 몰린다?)

**텐션 트라이앵글:**
- 제목: 정보 전달 -- "물량 넘기기" 의심 해소 + 블랙록이라는 권위 주체 + "진짜 이유"로 숨겨진 내막 암시

- 시너지 평가: 제목이 "물량 넘기기"를 정면 해소하고, 썸네일이 "전쟁인데 매수?"라는 시각적 충격을 주며, 이미지가 그 충돌을 비주얼로 확인시켜 "도대체 왜?"라는 클릭 충동 극대화. 레퍼 001의 검증된 [위기+기회 동시 자극] 공식 계승.

**Intro (158자, ~23초):**

## 패키지 B: "실적 시즌의 비밀"

### 컨셉
- 앵글: **미스터리 추적** -- "왜 하필 지금?" 스마트머니가 전쟁 리스크를 무시하고 매수에 나선 이유를 추적. 단서는 2주간 실적 발표 달력. JP모건 어닝 서프라이즈(EPS +9.19%) → 블랙록 반도체 80% 성장 전망 → 삼성전자 영업이익 +755% → 빅테크 AI capex $6,350억까지, 데이터 퍼즐 조각을 하나씩 맞추면 전체 그림이 보이는 구조.
- 핵심 약속: "다음 2주 실적 시즌에 돈이 어디로 흘러가는지 추적하면, 삼성전자·SK하이닉스가 왜 지금 움직이는지 소름돋게 이해하게 됩니다"
- 타겟 시청자: 삼성전자·SK하이닉스 주식을 보유하거나 매수 타이밍을 고민 중인 직장인 투자자. "반도체가 더 오르나?" 궁금하지만 전쟁 변수에 확신이 없는 사람.
- 서사 유형: **미스터리 추적** -- 단서(JP모건 실적) → 가설(실적이 전쟁을 이긴다) → 추가 단서(블랙록, 삼성전자, AI capex) → 진실(한국 반도체 메가 사이클). 정답이 나올 때까지 이탈 불가한 구조로 리텐션이 매우 강함.
- 감정 전략: **추리 쾌감 + 소름(연결의 깨달음)** / 의문("왜 지금?")(초반 15%) → 단서 발견의 작은 쾌감 반복(중반 45%) → 전체 그림 완성 소름(중후반 20%) → "내 계좌는?" 실행 의지(후반 20%) / 리텐션 접근: 미스터리 구조 자체가 이탈 방지(정답까지 못 떠남) + 한국 연결 블록에서 "내 돈" 스테이크 상승

## 패키지 C: "개미의 판단 기준"

### 컨셉
- 앵글: **소비자/직장인 관점의 판단 프레임** -- 월가나 기관이 아닌 "월급 300만 원 직장인" 시점에서 이 상황을 해석. 댓글 10위("월 330만 받으며... 자전거로 배달")의 서민 감정을 정면으로 끌어안으면서, "큰돈 없어도 쓸 수 있는 판단 기준"을 제공하는 앵글. 레퍼런스와 차별화된 새로운 방향.
- 핵심 약속: "월가 전문가 말 믿지 말라는 거 아닙니다. 이 영상 보시면 '어떤 숫자가 나오면 움직이고, 어떤 숫자가 나오면 멈추는지' 직장인용 체크리스트를 갖게 됩니다"
- 타겟 시청자: 투자하고 싶지만 "물리면 어쩌지"가 두려운 소액 직장인 투자자. 경제 유튜브를 보면서도 "결국 어쩌라고?"가 남는 사람. 매달 50만~100만 원 투자 여력인 사람.
- 서사 유형: **프로세스/여정** -- 단계별 판단 기준 제시(실적 체크 → 스마트머니 체크 → 리스크 체크 → 행동 기준). 다만 중반 이탈 위험이 높으므로, 각 단계마다 "이 숫자 하나면 충분합니다" 식의 단계 완료 보상을 삽입.
- 감정 전략: **자신감(판단 주체성 획득)** / 무력감("전문가만 아는 거 아니야?")(초반 15%) → "나도 읽을 수 있다" 작은 성공 반복(중반 45%) → 전체 체크리스트 완성 뿌듯함(중후반 20%) → "다음 주 이 숫자만 보면 됩니다" 실행 확신(후반 20%) / 리텐션 접근: 프로세스 서사의 높은 중반 이탈 위험을 "실전 체크리스트 완성"이라는 목표로 보완 + 각 체크포인트마다 "이것만 알면" 보상감 제공

### CTR

**제목 후보:**
1. `"이 숫자만 보세요" 다음 주 상승장, 매수 타이밍 잡는 직장인 체크리스트` (43자, 따옴표+하우투 패턴, "이 숫자만"으로 호기심 갭 + 직장인 타겟 명시 + 체크리스트로 실용적 보상)
2. `전쟁 끝나기 전에 사야 할까 말아야 할까? 블랙록·JP모건 데이터로 답 내림` (42자, 자유형식, 질문형+결과/증거 패턴, 시청자 실제 고민을 제목으로)
3. `"50만 원으로 뭘 하라고요?" 소액 투자자가 지금 봐야 할 3가지 숫자` (41자, 따옴표+숫자리스트 패턴, 서민 시점 공감 + 구체적 3가지로 정보적 보상 명확화)

**확정 제목:** 후보 1 -- `"이 숫자만 보세요" 다음 주 상승장, 매수 타이밍 잡는 직장인 체크리스트`
- 근거: 따옴표 문구 "이 숫자만 보세요"가 다른 경제 채널과 차별화되는 실용적 훅이면서 오픈루프(어떤 숫자?) 역할. "직장인 체크리스트"로 채널 타겟과 정확히 일치하는 시청자 셀프 셀렉션 유발. 43자로 모바일 최적. 하우투 패턴이 프로세스/여정 서사와 완전 호환.

**썸네일 텍스트:**
① `매수? 매도?` + `숫자가 답한다` (6자 + 7자 = 13자, 2줄 메인) -- L2 키워드 차용: 레퍼 001의 "확정형 선언" 톤을 차용하되 시청자 고민(매수/매도)을 직접 대변하는 구조로 변환.
② `D-7 체크리스트` + `지금 확인하세요` (9자 + 8자 = 17자, 2줄 메인) -- L1 구조 차용: 레퍼 001의 "전쟁 휴전 종료까지 D-7" + "정말 무섭게 오릅겁니다" [위기 키워드+행동 촉구] 구조를 그대로 차용, 핵심어를 "체크리스트"+"확인하세요"로 교체.
③ `직장인 투자` + `"이것만 보세요"` (6자 + 7자 = 13자, 2줄 메인) -- L4 감정 차용: 레퍼 001의 긴급함+확신 감정 톤 차용. 타겟을 직접 호명하여 셀프 셀렉션 + 따옴표로 인용감/권위 부여.

**이미지 컨셉:**
A. **체크리스트 + 빛나는 숫자** (화살표/그래프)
- 구도 패턴: 화살표/그래프
- 핵심 오브젝트: 체크리스트 보드(3개 항목, 체크 표시) + 핵심 숫자들이 빛나는 효과
- 색채/대비: 배경 짙은 네이비 + 포인트 금색(숫자 하이라이트)/녹색(체크 표시)
- 감정 톤: 확신(정돈된 판단 기준의 안정감)

B. **직장인 실루엣 + 갈림길** (인물 감정 + Before→After)
- 구도 패턴: 인물 감정 + Before→After(분기)
- 핵심 오브젝트: 정장 직장인 뒷모습 + 좌(빨간 하락 화살표)/우(녹색 상승 화살표) 갈림길
- 색채/대비: 배경 어두운 그레이 + 좌측 빨강/우측 녹색의 극적 색채 분할
- 감정 톤: 선택의 긴장(갈림길의 결정적 순간)

**텐션 트라이앵글:**
- 제목: 정보 전달 -- "이 숫자만 보세요"로 구체적 보상 약속 + 직장인 타겟 명시 + 체크리스트라는 실용적 산출물
- 썸네일 텍스트: 감정적 호기심 -- 매수/매도 갈림길의 긴장 + D-7 시간 압박
- 썸네일 이미지: 시각적 보완 -- 갈림길/체크리스트 비주얼로 "선택의 순간"을 시각화
- 시너지 평가: 제목이 "숫자"라는 구체적 보상을 약속하고, 텍스트가 매수/매도 선택의 긴장감을 건드리며, 이미지가 갈림길 비주얼로 "결정의 순간"을 강화. 세 요소가 합쳐져 "지금 내가 알아야 할 숫자가 있다"는 실용적 클릭 충동 유발. 다른 두 패키지와 완전히 다른 "실용/교육" 접근.
