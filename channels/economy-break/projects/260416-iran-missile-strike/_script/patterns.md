## 1. 반복되는 훅 공식

### 관찰

**훅 원문 (첫 문장)**:
> "지난주 사우디아라비아에서 현대 건설이 건설 중인 석유 화학 플랜트 현장 인근에 이란의 미사일이 떨어진 충격으로 사무실 천장 일부가 내려앉은 사실이 뒤늦게 알려졌습니다."

**훅 패턴 — "물리적 피해 압축 + 은폐 암시"**:
- 한 문장에 (1) 지정학 위기(이란 미사일), (2) 한국 대기업 연루(현대건설), (3) 물리적 피해(천장 붕괴)를 압축해 즉각적 위기감을 형성한다.
- "뒤늦게 알려졌습니다"라는 마감어가 **숨겨진 정보 존재**를 암시해 이탈 방지 역할을 한다. 이것은 순수한 사건 보도가 아니라 "왜 이제야 알려졌나?"라는 의문을 심는 장치다.
- 건드린 심리: 공포(미사일+붕괴) + 분노의 씨앗(은폐 암시) + 정보 비대칭("나만 모르고 있었나")

**인트로 원문**:
> "현장 직원들은 회사가 제대로 된 대피 매뉴얼도 마련하지 않았다며 불안해하고 있습니다. 사측은 재택 근무을 비롯해 안전의 최선을 다하고 있다는 입장을 내놨습니다."

**인트로 패턴 — "갈등 구도 즉시 투입"**:
- 훅의 충격 직후 "직원 vs 회사"의 대립 프레임을 설정한다. "매뉴얼도 없다"(직원 불안) vs "최선을 다하고 있다"(회사 입장)의 대비가 "누가 맞는지"를 판단하기 위한 시청 동기를 만든다.
- "오늘은 ~에 대해 알아볼게요" 같은 메타 설명 없이 바로 갈등으로 진입해 효과적이다.

### 설계 원칙

1. **첫 문장에 3요소 압축**: 지정학 위기 + 한국 기업/직장인 연결 + 구체적 물리 피해를 한 문장에 넣어 "이건 남의 일이 아니다"를 즉각 인식시킨다.
2. **"뒤늦게/몰래/숨겨진" 프레이밍 삽입**: 첫 문장 안에 정보 은폐·지연 암시를 넣어 "더 있다"는 기대감을 건다.
3. **인트로에서 갈등 구도 즉시 설정**: 훅 직후 "피해자 주장 vs 가해자(기업/정부) 입장"을 대비시켜 시청자가 편을 들게 만든다. 메타 설명("오늘은~") 절대 금지.
4. **훅 20초 이내 완결**: 레퍼런스가 2분 14초 뉴스에서도 훅+인트로를 2문장으로 끝냈다. 우리 영상(10~15분 예상)에서도 훅+인트로를 30초 이내로 압축한다.

## 2. 반복되는 본론 구조 & 전달 기법

### 관찰

**구조 — "고발 보도 5단계"**:

| 단계 | 내용 | 역할 |
|------|------|------|
| 1. 현장 영상 묘사 | 숙소 앞 미사일 화염, 4km 거리 피격 | 시각적 공포 확립 |
| 2. 물리적 피해 확인 | 7일 건물 흔들림, 천장 일부 붕괴 | 위협이 실제였음을 입증 |
| 3. 긴장 지속 | 종전 협상 소강 → 결렬 시마다 불안 가중 | 안도를 허용하지 않고 위기 유지 |
| 4. 피해자 증언 | 익명 직원 "매뉴얼 없다, 노트북 지참 퇴근뿐" | 분노 폭발점 — 기업 무책임 폭로 |
| 5. 기업 반박 | "24시간 비상 상황실, 복귀 희망자 귀국 지원" | 대비 효과로 허술함 부각 |

핵심: **사건 → 피해 입증 → 긴장 유지 → 내부자 폭로 → 공식 반박** 순서. 이 순서가 감정을 점진적으로 끌어올린다. 특히 4단계(폭로)를 후반에 배치해 분노의 피크를 영상 후반부에 만든다.

**전달 기법**:

- **시간순 서사**: 8일 영상 → 7일 천장 붕괴 → 13일 협상 결렬 이후. 날짜를 구체적으로 찍어 "실제 타임라인"을 느끼게 한다.
- **증언 직접 인용**: > "위험 상황을 알리는 사우디 국가 경보에도 회사는 어떤 대피 안내도, 행동 매뉴얼도 마련해 놓지 않고 있다" — 3인칭 서술이 아닌 당사자 목소리로 신뢰감과 분노를 동시에 확보.
- **숫자는 거리·날짜로 현장감 부여**: "숙소에서 4km 가량", "8일", "13일" 등 구체적 수치가 "실제 상황"임을 각인시킨다.
- **대비 구조**: 직원 증언(무대응) vs 회사 입장(24시간 비상 상황실) — 두 입장을 나란히 놓아 시청자가 스스로 판단하게 유도하되 실제로는 기업의 허술함이 부각되도록 설계.

**리텐션 장치**:

- **오픈루프 (중반)**: > "다행히 그 이후 이란의 종전 협상이 시작되며 공격은 잠잠해졌지만 협상 결렬 소식이 전해질 때마다 현장 직원들의 불안감은 커지고 있습니다" — 소강 상태를 안도로 닫지 않고 "하지만"으로 재불안 루프를 유지. 안심→재위기 리듬이 효과적.
- **전환 멘트 (중반→후반)**: "JTBC와 전화 통화가 연결된 한 현장 직원은~" — **익명 내부자 등장 예고**로 폭로 기대감 유도. 화자가 바뀌는 순간 집중력이 리셋된다.
- **패턴 인터럽트 (후반)**: > "노트북을 지참해 퇴근하라는 지시 외에는 별다른 대응이 없었다" — 추상적 비판이 아닌 "노트북 지참 퇴근"이라는 구체적이고 황당한 디테일로 분노를 환기.

### 설계 원칙

1. **본론 뼈대는 "고발 5단계" 기반**: 현장 충격 → 피해 실증 → 긴장 유지 → 내부자 폭로 → 공식 입장 대비. 우리 영상은 10~15분이므로 각 단계를 2~3분씩 할당하되, 폭로 단계를 가장 길게(3~4분) 잡아 분노 피크를 만든다.
2. **날짜·거리·금액은 반드시 구체 수치로**: "최근" "가까운 거리" 같은 모호 표현 대신 "4월 7일", "4km", "수주 잔고 XX조원" 등 숫자로 현장감을 부여한다.
3. **증언은 직접 인용체로**: 3인칭 요약("직원들이 불안해했다") 대신 1인칭 인용("매뉴얼도 없이 어떡하라는 건지 모르겠다") 형식을 사용한다.
4. **대비는 항상 쌍으로**: 기업 발표/약속 vs 현장 실태를 나란히 놓아 시청자가 괴리를 직접 체감하게 한다.
5. **안도 금지 — "하지만" 전환**: 소강 상태나 긍정 정보를 제시한 직후 반드시 "하지만/그런데" 전환으로 위기감을 재점화한다.
6. **파트 전환 직전 오픈루프**: 다음 파트의 핵심 폭로를 암시하는 문장을 파트 경계에 배치한다.
7. **패턴 인터럽트는 구체 디테일로**: 추상적 비판("대응이 미흡했다") 대신 황당한 구체 사실("노트북 지참 퇴근이 전부")을 삽입해 감정을 리셋+증폭한다.

| 분노 | "노트북을 지참해 퇴근하라는 지시 외에는 별다른 대응이 없었다" — 기업 무책임의 구체적 증거 |

**감정 흐름**: 공포(초반 30%, 현장 영상 묘사) → 분노(중반~후반 50%, 매뉴얼 부재 + 기업 무대응 폭로) → 경각심(후반 20%, 현재진행형 위기 강조)

### 설계 원칙

1. **분노가 1순위 감정**: 댓글에서 가장 많은 공감을 얻은 감정이 분노이므로, 우리 영상의 감정 설계 중심축은 분노다.
2. **분노의 이중 타깃 설정**: (1) 국제정치 원인(전쟁을 일으킨 세력), (2) 기업의 무책임(직원 안전 방치). 두 타깃을 번갈아 자극해 분노 지속시간을 늘린다.
3. **공포로 기초 깔기 → 분노로 전환**: 초반 2분 이내에 현장 위험 묘사(오감 동원: 화염, 폭발음, 건물 진동)로 공포를 충분히 확립한 후, "그런데 회사는?" 전환으로 분노로 넘긴다.
4. **구체적 디테일로 분노 트리거**: "대응이 미흡했다"(추상) 대신 "노트북 지참 퇴근이 유일한 조치"(구체) 같은 황당한 팩트를 분노 폭발점으로 사용한다. 이런 구체 디테일을 최소 3개 이상 확보한다.
5. **"나에게도 해당된다" 경각심으로 마무리**: 후반부에 한국 건설사 중동 파견 규모, 한반도 위기 가능성 등을 제시해 시청자 자기 투영을 유도한다.
6. **감정 흐름 설계**: 공포(초반 20%) → 분노 축적(중반 50%) → 분노 폭발(후반 전반 20%) → 경각심(후반 마무리 10%)

- 모바일 가시성: 현장 캡처의 붕괴 흔적은 160x90px에서 인식 어려움. 반면 대형 텍스트 + 기업 로고는 식별 가능. **모바일에서는 텍스트+로고가 클릭 유도의 주역**.

**썸네일 비주얼 추가 분석**:
- 현장 영상 캡처(PIP 형태)는 "실제 상황"이라는 진정성을 부여. 그래픽이 아닌 실제 촬영 이미지가 신뢰감을 높인다.
- 어두운 배경 위 밝은 대형 텍스트 구조는 스크롤 중 시인성이 높다.

**제목 분석**:
- 제목: "미사일 충격에 천장 내려앉았다…현지 직원들 패닉"
- 구조: **[물리적 결과] + 말줄임표 + [감정 반응]** — 결과 스포일러형 + 감정 명시형 복합
- 말줄임표(…)가 "그 다음에 뭐가?" 긴장감을 유지하며, "패닉"으로 감정 강도를 명시
- 기업명(현대건설)을 제목에서 의도적으로 제외 → 썸네일 로고로만 노출. 클릭 시 "어느 기업이지?" 궁금증이 추가 작동.

**썸네일-제목 역할 분담**:
- 썸네일: **감정(공포) + 기업 식별(현대건설 로고)** — 시각적 충격 전달
- 제목: **사건 요약(미사일→천장 붕괴) + 현장 반응(패닉)** — 정보 전달
- 역할 분담 공식: 썸네일이 "누구에게" + "어떤 감정", 제목이 "무슨 일이" + "얼마나 심각한지"

### 설계 원칙

1. **썸네일 텍스트 — 2줄 계층형, 총 20자 내외**:
   - 첫째 줄: 사건/결과를 물리적 동사로 (8~12자). "천장 내려앉아", "공장 멈춰섰다" 같은 시각화 가능한 표현.
   - 둘째 줄: 감정 반응 또는 파급 효과 (6~8자). "직원들 패닉", "국내 기업 비상" 등.
   - 기업명/인물명은 로고 또는 별도 요소로 분리 배치.
2. **제목 — [물리적 결과…감정/상태] 구조, 40자 내외**:
   - 구체적 물리 피해("천장 내려앉았다")로 시작.
   - 말줄임표(…)로 긴장 유지.
   - 인물/집단의 감정 상태("패닉", "비상", "충격")로 마무리.
3. **역할 분담 — 썸네일=감정+식별, 제목=사건+심각도**:
   - 썸네일 텍스트와 제목이 동일 문구를 반복하지 않도록 정보를 분배한다.
   - 기업명/인물명은 썸네일에만, 사건 경위는 제목에만 담는다.
4. **썸네일 비주얼 — 어두운 배경 + 현장 이미지 + 밝은 대형 텍스트**:
   - 배경은 현장 캡처 또는 관련 이미지를 어둡게 처리.
   - 하단 35%에 2줄 대형 텍스트 블록 배치.
   - 기업 로고/인물 사진을 우측에 오버레이해 "누구 이야기인지" 즉시 인식시킨다.
   - 모바일 160x90px에서 텍스트와 로고가 식별 가능한지 반드시 검증.
