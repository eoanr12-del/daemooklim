## 패키지 A: "오르반의 시대가 끝났다는 신호 3가지"

### 컨셉
- 앵글: 선거 전 불안감을 "이미 끝난 신호"로 재해석. 시스템 메커니즘(기울어진 운동장) 분석으로 여론 역전의 불가피성 증명.
- 핵심 약속: 이 영상을 끝까지 보면, 여론조사 수치 이전에 이미 오르반 체제가 무너지고 있었다는 냉정한 진실을 알게 되면서, 동시에 제도적 유산이 얼마나 위협적인지 깨닫게 된다.
- 타겟 시청자: 경제 악화로 지쳐 있는 30대 직장인. "정치 좋아하지만 결국 경제 문제 아닌가"라고 생각하는 사람.
- 서사 유형: 시간순 서사 (1989년 → 16년 조종 → 현재 균열 신호 → 불확실한 미래) - 시간축과 규모축을 동시에 확장하여 거대한 체제 붕괴의 필연성 보여주기. 레퍼런스와 동일 구조로 검증된 방식 활용.
- 감정 전략: 목표감정 "냉정한 분노" (분노가 희망으로 전환되지 않도록) / 감정 아크 충격(0-1분) → 분노(2-5분, 부패 스캔들 강조) → 불안(5-8분, 제도적 유산 상기) → 결말 열림 / 리텐션 접근 오픈루프 3개: "대체 어떻게 이런 체제가?" (1분) → "경제 악화와 부패의 악순환" (4분) → "총리 바뀐다고 끝날까?" (7분)

### CTR
- 제목 후보:
  ① `"이미 끝났다" 신호 3가지, 헝가리 오르반이 놓친 것`
  ② `헝가리 총리 오르반, 여론조사보다 먼저 보인 패배의 신호들`
  ③ `오르반 16년, 경제 붕괴로 읽는 독재 체제의 필연적 몰락`
- 확정 제목: `"이미 끝났다" 신호 3가지, 헝가리 오르반이 놓친 것` (40자)

  - 따옴표 + 수치 조합으로 구체성 강화
  - 시간순 서사 구조와 호환성 ✓

- 썸네일 텍스트:
  ① `경제 붕괴` + `부패 스캔들` + `여론 역전` (각 3~4줄, 계층형)
  ② `물가 17.6%` / `집값 290%` / `이미 끝났다` (데이터 중심)
  ③ `EU제재 1순위` / `언론자유 68위` / `오르반의 끝` (국제적 위상 강조)

  - **A. 검정 배경 + 빨간 아래향 화살표**: 오르반의 얼굴 오른쪽에 추락 이미지 (구도: 오르반 60% + 배경 40%), 색상은 어두운 톤에 빨강/주황 강조, 감정톤은 "돌이킬 수 없는 하락"
  - **B. 경제 지표 차트 배경 + 오르반 옆모습**: 뒷배경에 물가/집값 상승 그래프, 오르반은 걱정스러운 표정, 색상은 회색/파랑 톤에 노랑 강조선, 감정톤은 "경제적 책임감"

- 텐션 트라이앵글:
  - **제목**: 명확한 수치 ("3가지") + 선명한 메시지 ("이미 끝났다") → 호기심
  - **텍스트**: 부패/경제 데이터의 냉정함 → 분노 감정 유도
  - **이미지**: 하락/붕괴 시각 + 무겁고 어두운 톤 → 불안 증폭
  - **시너지**: "모르던 위협"(제목) + "측정할 수 있는 붕괴"(텍스트) + "시각적 낙하"(이미지) = 합리적 분노 자극

## 패키지 B: "오르반 패배, 선거 승리보다 먼저 온 신호"

### 컨셉
- 앵글: 2026.04.12 선거 결과를 기반으로 **이미 일어난 패배를 역추적 분석**. 여론조사보다 먼저 나타난 "공식적 패배 선언" 신호들을 포착. 레퍼런스와의 핵심 차별점: 선거 후 현황 영상.
- 핵심 약속: 이 영상을 끝까지 보면, 오르반이 53.6% 대 37.8%로 패배한 이유가 단순한 "여론 변동"이 아니라 체제 자체의 자가붕괴였음을 알게 되고, 동시에 "그래도 제도는 남았다"는 불편한 진실을 마주하게 된다.
- 타겟 시청자: 정치/경제 뉴스를 실시간으로 팔로우하는 40대 회사원. "결국 누가 지배하든 변하지 않는 건 변하지 않는다"고 생각하되, 그래도 시스템의 변화가 중요하다고 느끼는 사람.
- 서사 유형: Before→After 대비 + 불완전한 결말 - "선거 전 예상"(레퍼런스) vs "선거 후 현실"(우리 영상)의 대비로 시청자의 예상과 현실을 충돌시켜 리텐션 극대화. 최종 메시지는 "이제 시작일 뿐"으로 열린 구조 유지.
- 감정 전략: 목표감정 "안도감과 우려의 혼재" / 감정 아크 안도(0-2분, 오르반 패배 확정) → 의심(2-5분, 대안의 신뢰도 검증) → 불안(5-8분, 제도적 복잡성) → 미지의 기대 / 리텐션 접근 패턴 인터럽트: "패배했는데 왜 끝이 아닐까?"로 시청자 기대치 뒤집기 (중반) → 오픈루프로 마무리 "Tisza당이 정말 바꿀 수 있을까?"

### CTR
- 제목 후보:
  ① `오르반 패배 확정, 그런데 진짜 문제는 지금부터다`
  ② `"53.6% vs 37.8%" 오르반의 공개 패배와 헝가리의 불안한 미래`
  ③ `선거 2일 전과 다른 이유, 헝가리가 오르반을 버린 진짜 신호`
- 확정 제목: `오르반 패배 확정, 그런데 진짜 문제는 지금부터다` (35자)
  - 모바일 잘림 안전
  - "패배" + "그런데" (반전) → 개방형 구조로 호기심 유지
  - Before→After 구조와 호환성 ✓

- 썸네일 텍스트:
  ① `오르반 패배 확정` + `53.6% vs 37.8%` + `문제는 지금부터` (숫자 강조)
  ② `선거 결과` / `Tisza 당 승리` / `오르반의 끝` (명확한 결과 제시)
  ③ `여론 역전` / `제도적 유산` / `민주주의는?` (질문형으로 미스터리 유지)

- 이미지 컨셉:
  - **A. 스플릿 스크린 (좌/우)**: 왼쪽 오르반의 찡그린 표정 + 빨강, 오른쪽 마자르의 밝은 표정 + 파랑, 중앙에 51% vs 49% 비율 표시 (구도: 대비), 감정톤은 "명확한 승패"
  - **B. 오르반 아래향 화살표 + 배경 민주주의 상징 (투표함)**: 오르반 우측에 하락 곡선, 배경에 EU 깃발과 투표함 (구도: 시스템 전환), 색상은 회색→파랑 그레이디언트, 감정톤은 "체제 전환의 무게"

- 텐션 트라이앵글:
  - **제목**: "패배 확정"(기대 충족) + "그런데"(반전) + "지금부터"(열린 구조) → 혼합 감정
  - **텍스트**: 명확한 수치(53.6%) + 불안의 신호("문제는") → 안도와 우려 동시
  - **이미지**: 명확한 승패 비주얼 + 시스템 전환의 무게감 → 감정적 복합성
  - **시너지**: "완승 인정했네"(제목 1차) → "그런데?"(제목 2차) → "역시 복잡하네"(텍스트/이미지) = 예상 깨기 → 다시 집중

## 패키지 C: "물가 17.6%가 독재자를 내쫓은 방법"

### 컨셉
- 앵글: 선거 결과를 소비자/일반인의 "생활고" 관점에서 역추적. "정치"가 아닌 "경제"가 선거를 판결했다는 명확한 인과관계 제시. 3개 패키지 중 가장 개인화된 앵글.
- 핵심 약속: 이 영상을 끝까지 보면, 당신이 느끼는 물가 상승과 집값 상승이 단순한 "경제 지표"가 아니라 국제 정치의 축을 바꿨다는 현실감을 갖게 되고, 동시에 "경제가 민주주의의 최후의 보루"라는 냉정한 진실을 깨닫게 된다.
- 타겟 시청자: 자영업자나 저금리/고물가로 힘들어하는 20~30대. "정치는 복잡한데 경제는 명확하다"고 생각하는 사람. 한국의 물가/집값과 헝가리를 직접 비교하며 공감하려는 사람.
- 서사 유형: 비교/대결 (헝가리 vs 한국 물가/집값) + 인과 추적 (경제 악화 → 선거 판결) - "헝가리 물가 17.6% = 당신의 현실"이라는 직접적 연결. 또는 미스터리 추적 (왜 여론조사는 이리 뒤늦게 움직였나? → 경제 절박감의 누적)
- 감정 전략: 목표감정 "공감과 분노의 혼합" / 감정 아크 공감(0-2분, "나도 물가 올랐는데") → 분노(2-5분, "이게 정책이야?") → 희망(5-6분, "그래도 바뀔 수 있네") → 무거움(6-8분, "하지만 우리는?") / 리텐션 접근 개인화 질문: "당신의 물가는 몇 % 올랐어요?" (0분) → "헝가리 자영업자들은 뭘 했을까?" (3분) → "우리도 같은 무기를 가지고 있어요" (6분)

### CTR
- 제목 후보:
  ① `물가 17.6%, 이것이 독재자를 내쫓은 방법`
  ② `당신의 물가 올라가는 이유, 헝가리 선거에 있다`
  ③ `집값 290% 상승 + 물가 17.6%, 선거로 광기를 멈춘 나라`
- 확정 제목: `물가 17.6%, 이것이 독재자를 내쫓은 방법` (32자)
  - 모바일 잘림 안전
  - 데이터(17.6%) + 인과(독재자 내쫓은) → 직관적 연결
  - 비교/대결 구조와 호환성 ✓

- 썸네일 텍스트:
  ① `물가 17.6%` + `집값 290%` + `선거로 끝` (경제 수치 중심)
  ② `생활물가` / `자영업 붕괴` / `오르반 퇴출` (소비자 관점 명시)
  ③ `당신의 경험이` / `헝가리 선거였다` / `한국은?` (개인화 + 질문)

- 이미지 컨셉:
  - **A. 오르반 vs 슈퍼마켓 물가표**: 왼쪽 오르반의 거부하는 손짓, 오른쪽 물가 상승 기호(↑) + 음식/집값 아이콘, 색상은 빨강(물가) vs 검정(거부), 구도는 명확한 대조, 감정톤은 "경제적 절망"
  - **B. 투표함과 장바구니 (오버레이)**: 투표함에 장바구니가 겹쳐있고, 오르반의 얼굴이 어두워지는 구도, 색상은 초록(돈/경제) + 검정(정치), 감정톤은 "경제가 정치를 결정한다"

- 텐션 트라이앵글:
  - **제목**: 구체적 수치(17.6%) + 강한 인과(내쫓은) → 호기심 + 공감
  - **텍스트**: 소비자 용어(생활물가, 자영업) + 개인화 질문 → 감정적 몰입
  - **이미지**: 경제 기호 + 정치적 결과의 명확한 연결 → 인과관계 시각화
  - **시너지**: "내 지갑이 정치다"(제목) + "당신도 투표처럼"(텍스트) + "경제=권력"(이미지) = 소비자의 정치적 권능감

### 패키지 B Hook & Intro

**Hook (42자, ~8초)**:
53.6% vs 37.8%. 오르반이 이 숫자를 보고 공개적으로 패배를 인정했습니다. 그런데 끝이 아니라 시작이거든요.

**Intro (168자, ~19초)**:
오르반은 16년 동안 선거 제도를 조종했어요. 언론을 장악했어요. 사법부까지 통제했어요. 말 그대로 운동장을 기울였거든요. 그런데 이렇게까지 준비했는데도 패배했다니, 뭔가 이상하잖아요. 이 영상에서는 왜 그렇게까지 확실한 패배가 왔는지, 그리고 패배 후에도 남아있는 불안한 것들이 뭔지를 확인해볼 거예요.

### 패키지 C Hook & Intro

**Hook (41자, ~7초)**:
당신의 물가, 올랐잖아요. 헝가리 시민들은 이 물가 상승으로 선거장에 갔습니다. 그리고 독재자를 내쫓았거든요.

**Intro (172자, ~20초)**:
17.6%. 한국도 최근에 겪은 물가 상승이죠. 헝가리는 2023년에 이 정도를 감수했어요. 집값도 290% 올랐어요. 자영업자들은 장을 담글 수 없게 됐어요. 그렇게 경제가 무너지면서 시민들은 투표장으로 갔고, 16년을 지배하던 독재자를 한 번에 내쫓았어요. 당신이 느끼는 물가 상승이 사실은 얼마나 강력한 정치적 무기인지, 이 영상에서 확인해보세요.
