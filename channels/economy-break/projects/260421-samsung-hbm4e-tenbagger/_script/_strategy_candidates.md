- **앵글**: **반도체 메가 사이클이 직장인 월급·성과급·집값·취업 시장에 실제로 어떻게 번지는지, 피크(2027년 여름)가 오기 전에 무엇을 준비해야 하는지 소비자 관점으로 풀어내는 실생활 파급 해부**. 001에서 이형수가 흘린 "SK하이닉스 직원 10억 성과급", 부동산·외제차 이야기를 전면에 끌어올려 — "하닉고시" 열풍, 취업 선호도 1위 역전, 성과급 7억~13억, K-칩스 세제 논쟁까지 직장인 체감 포인트로 재구성.
- **핵심 약속**: "삼성이 1분기에 57조를 벌었다는 뉴스가 내 월급·성과급·집값·우리 동네에 구체적으로 어떻게 번지는지, 그리고 2027년 피크 전에 뭘 해야 하는지 직장인 관점으로 깨닫게 됩니다."
- **타겟 시청자**: **반도체 업계가 아니지만 "내 월급이 왜 안 오르지" 답답해하는 20~40대 비반도체 직장인** + **취업·이직 고민하는 2030 젊은 직장인** (002 댓글 "강제로 투자했음 일만 열심히 하면 되는데" 좋아요 4에 정면 대응 + 리포테라 "하닉고시" 열풍 신규 데이터 활용)
- **서사 유형**: **N포인트 해부 (실생활 파급 4~5개)** — 단, 채널 선호인 "반전 서사" 프레임을 얹어 "반도체 호황 = 남의 일 → 사실은 내 일"이라는 반전으로 진행. 중반 이탈 위험 높음 (N포인트) → 각 포인트를 "내 월급", "내 성과급", "내 집값", "내 취업"의 4막 구성으로 리셋 방지.
- **감정 전략**:
  - 목표 감정: **통쾌 + 불안 (FOMO) 혼합** — 채널 타겟(직장인)이 "반도체 뉴스 = 남의 일" 치부하다가 "내 일이었네" 깨달으며 느끼는 정서
  - 감정 아크: **무관심(반도체 뉴스 = 남의 일) → 놀람(SK 성과급 7억, 하닉고시 열풍) → 공감(내 회사도 영향받음) → 위기감(2027 피크 전 준비) → 결단(지금 뭘 할지)**
  - 리텐션 접근: 4개 파급 포인트를 오픈루프 선언("여러분 월급·성과급·집값·취업 네 가지가 모두 바뀝니다") → 각 파트마다 시청자 속마음 대역 질문 ("그럼 내 회사는요?")

### CTR

- **확정 제목**: `"성과급이 10억이라고요?" 직장인 월급·집값을 뒤흔들 삼성 57조의 진짜 파급` (45자, 따옴표 형식)
- **제목 후보**:
  1. `"성과급이 10억이라고요?" 직장인 월급·집값을 뒤흔들 삼성 57조의 진짜 파급` (45자, 따옴표+실생활)
  2. `하닉고시 열풍, 성과급 7억, K-칩스 논쟁 — 반도체 메가사이클이 직장인에게 미치는 4가지` (51자, 숫자 리스트 패턴)
  3. `삼성이 1분기에 57조 벌어서 내 월급이 오를까? 반도체 호황의 진짜 수혜자 분석` (42자, 질문형+결과)
- **썸네일 텍스트**:
  ① `"성과급 10억"` + `남의 일 아닙니다` + `월급·집값·취업` + `이코노미브레이크` (FOMO 유발)
  ② `"내 월급도" 오를까?` + `삼성 57조의 낙수` + `직장인 필독` + `이코노미브레이크` (공감 유도)
  ③ `"하닉고시" 현상` + `취업 판도 바뀐다` + `삼성·SK 역전` + `이코노미브레이크` (뉴스성 반전)
- **이미지 컨셉**:
  - **A. 스케일 대비 + 일상 오브젝트**: 거대한 삼성/SK 반도체 공장(배경) → 앞쪽에 작게 배치된 "월급 명세서 / 아파트 / 외제차 / 취업 합격증" 4개 오브젝트가 빛나며 쏟아지는 구도. 어두운 배경 + 금색·빨강 포인트. 추상 "반도체 호황"을 일상 이미지로 환원.
  - **B. 인물 감정(직장인 반응) 클로즈업**: 정장 입은 20·40대 직장인이 성과급 명세서/청약 당첨 알림을 보며 놀라는 표정 + 배경에 희미하게 삼성/SK 로고 + "7억", "10억" 숫자가 떠있음. 002 인물 감정 구도 L1 차용, 권위자(젠슨황) → 일반 직장인으로 교체.
- **텐션 트라이앵글**:
  - 제목 = "성과급 10억" **숫자 충격 + 직장인 관점 확정**
  - 썸네일 텍스트 = "남의 일 아닙니다" **공감 프레임 + 따옴표 도발**
  - 이미지 = 직장인 놀라는 표정 / 일상 오브젝트 쏟아짐 **실생활 시각화**
  - 시너지: "반도체? 난 관심없는데" → "내 월급?!" 전환 클릭 유도. 3요소가 "남의 일 → 내 일" 반전을 합작.

### Hook & Intro

- **Hook (78자)**: "SK하이닉스 직원들이 올해 평균 7억, 최대 13억까지 성과급을 받을 거라는 이야기, 들어보셨나요?"

**근거 (2문장)**:
1. **채널 선호 서사 + 최고 퍼포먼스 데이터 정합성**: profile.md 선호 "몰락 서사" + 002(24k, 애국 자긍심 댓글 좋아요 28·18·14)의 "2년 암흑기 → 부활" 척추 구조 + 001(65k) "과거 증거 → 현재 숫자 대비쌍" 훅 공식을 모두 포괄. 시간순 서사는 중반 이탈 위험이 가장 낮고(15분 러닝타임에 유리) "그 다음은?" 궁금증이 자연스럽게 유지됨.
2. **핵심 팩트의 임팩트 순위 1위 활용**: 검증된 "2026 1Q 57조 vs 2018 연간 58.89조" 데이터 대비쌍은 verified-data.md에서 가장 강한 숫자 충격(시급 265억원/시간 환산까지 가능). 002에서 "한 분기 57조" 언급이 댓글 1위(좋아요 28)를 직접 만들어낸 트리거였음이 추적 가능. 추가로 전영현 복귀·3개 스톤·평택 5공장 등 파생 서사를 15분 분량에 깔끔하게 담을 수 있음.

**대안 검토**:
- 패키지 B(역발상)는 CTR 잠재력은 가장 높으나, "SK가 못 한 걸"이 경영진/SK 팬덤을 자극할 리스크 + 기술 디테일(4나노/12나노) 설명에 15분 중 40%가 소요될 가능성 → 직장인 점심시간 톤(profile.md)과 다소 괴리.
- 패키지 C(실생활 파급)는 채널 관점(소비자/직장인)에 가장 적합하나, "제목에 낚였네"(001 좋아요 2) 리스크 회피 위해서는 삼성 57조·HBM4 본 서사가 전면에 와야 하는 주제 특성상 부제 수준으로 흡수하는 것이 안전. → 패키지 A의 후반 30%(70~90% 구간)에 패키지 C의 실생활 파급 4개 포인트를 녹여 쓰면 두 장점 결합 가능.
