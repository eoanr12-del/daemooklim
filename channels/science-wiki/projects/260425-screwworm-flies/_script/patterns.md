## 1. 반복되는 훅 공식

**관찰**:
- 레퍼런스 훅 원문: "파나마 지역 위를 비행기가 날고 있는데요. 꼬리 쪽을 보시면 뭔가를 흩뿌리고 있습니다. 농약을 살포하는 건가 싶을 텐데 사실 이건 모두 파리입니다."
- 패턴: **시각적 장면 묘사 → 자연스러운 추측 유도 → 정답 반전 → 충격 숫자**
- 효과: "농약 → 파리"의 의외성으로 즉시 패턴 인터럽트. 시청자가 머릿속에 그림을 그린 직후 정답이 뒤집히면서 호기심 폭발.
- 인트로 패턴: 훅 직후 **귀여운 대상(키 사슴) → 끔찍한 묘사(두개골 노출) → 살아있다는 충격** 3단으로 정서적 미스터리 증폭.

**설계 원칙**:
1. **첫 5초 안에 시각적 장면 1개 + 의외성 반전 1개**를 박는다. (예: "비행기에서 농약 같은 걸 뿌려요. 근데 이게 농약이 아니에요")
2. 훅 직후 **공포·혐오 디테일 1개**를 짧게 박아 본문 진입까지 호기심을 유지한다. (예: 키 사슴 두개골 묘사)
3. **본문 진입 직전에 핵심 약속 질문 2개를 명시적으로 던진다.** ("이 파리의 정체는 뭐고, 왜 미국이 그걸 일부러 뿌리는 걸까요?")
4. 잡학위키 톤(친근 존댓말)으로 변환: "~인데요" 금지 → "~거든요/~잖아요" 사용.

## 2. 반복되는 본론 구조 & 전달 기법

**관찰**:

*구조*: **충격 사건 → 정체 공개 → 공포 누적 → 박멸 영웅담 → 위기 재발 → 미래 기술 → 내 식탁 (남 일 아님)** — 7단 시간순 + 영웅서사 + 클리프행어 결합.

*전달 기법*:
- **비유**: "나사 모양처럼 몸을 비틀면서", "마치 빗자루로 쓸어내듯이 파리들을 국미 대륙의 끄트머리까지", "엑스레이 장비를 몰래 빌려다가" — 메커니즘과 작전 규모를 이미지로 변환.
- **숫자**: 200~400개 알 / 12시간 부화 / 매주 1억 마리 / 주 100톤 고기 / 3.8만L 피 / 매년 수조원 / 2023년 9,300건 / 미국산 소고기 47% / 53% 인상 — 작은 단위와 거대 단위를 교차 배치.
- **사례**: 1858년 악마의 섬 죄수 / 2016년 키 사슴 / 텍사스 미션 공장 / 2006년 파나마 공장 / 2024년 멕시코 북상 — 인물·장소·연도가 모두 구체.

*리텐션 장치*:
- 오픈루프(초반): "도대체 이 파리의 정체가 뭐길래... 과학자들은 왜 파리떼를 쏟아붙는 기묘한 일을 벌이고 있는 걸까요?"
- 미스터리 감추기(중반): "결정적인 단서 하나를 포착했는데요" — 정답을 한 박자 늦춰 보여줌
- 위기 전환(후반): "그러던 2022년 이 방어 시스템에 균열이 생기기 시작했습니다" — 박멸 성공 안도 후 다시 끌어올리기
- 충격 디테일(전체): "두개골이 다 보일 정도로", "항공사들이 파리 운송을 거부", "작업자들이 향수를 뿌려가며"

**설계 원칙**:
1. **본론은 시간순 + 영웅서사 결합 7단 구조**를 그대로 채택. (사건→정체→공포→영웅→위기→기술→내 식탁)
2. **숫자는 1~2분에 1번씩 의무 배치**, 작은 수치(알 200개, 12시간)와 거대 수치(주 1억 마리, 수조원)를 교차해 스케일 감각을 흔든다.
3. **비유는 추상 메커니즘 등장 시마다** (불임 메커니즘, 유전자 드라이브) 일상 이미지로 즉시 변환. 잡학위키 톤이라 "쉽게 말하면요" 시그니처를 활용.
4. **파트 전환 직전 오픈루프 의무**: 다음 파트 정답을 한 줄로 미루는 문장 삽입.
5. **충격 디테일 보존**: 두개골 묘사, 향수 일화, 주 100톤 고기 등은 잡학 임팩트의 핵심이라 톤 다듬되 디테일은 유지.

*감정을 만든 장치 역추적*:
- 신기함: "인류가 최초로 곤충의 성생활을 역이용해서 해충 박멸에 성공한 역사적인 순간"
- 공포·혐오: 키 사슴 두개골 묘사, 1858년 악마의 섬 죄수 사례
- 안도→불안 재전환: "방어 시스템에 균열이 생기기 시작했습니다"

*감정 흐름*: **공포(훅·정체) → 안타까움(피해) → 감탄(박멸 영웅) → 안도(성공) → 불안 재발 → 우려(내 식탁)**

**설계 원칙**:
1. **공포·혐오는 초반 2분 안에 정점 1회**: 키 사슴 사례 + 악마의 섬 죄수 사례를 짧게 임팩트만.
2. **중반 4~7분은 감탄/통쾌**: 니플린 박사 발견 → 부시랜드 합류 → 퀴라소 0% 부화 → 멕시코 빗자루 — 영웅서사로 카타르시스.
3. **후반 9~12분은 불안 재점화**: 코로나·밀수로 방어망 붕괴 → 멕시코 진출 → 한국 식탁.
4. **클로징은 우려 + "관심 가지고 지켜보자" 톤**: 잡학위키의 "내 삶에 무슨 상관?" 관점 강조.
