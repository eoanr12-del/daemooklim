## 패키지 A: "사상 최대 규모의 거세 작전" — 영웅 서사 + 미친 디테일

### 컨셉
- **앵글**: 미국이 70년 동안 매주 수억 마리의 파리를 거세시켜서 비행기로 뿌려 온 인류 역사상 가장 황당하고 위대한 박멸 작전. 두 명의 곤충학자(니플린·부시랜드)가 X선 장비를 몰래 빌려서 시작한 무모한 실험이 어떻게 60년에 걸쳐 미국→파나마까지 빗자루질을 해냈는지 영웅서사로 풀어낸다.
- **핵심 약속**: "곤충 거세"라는 황당한 단어 하나로 시작하는 잡학 폭격. 술자리에서 "야 너 그거 알아?"로 30분은 우려먹을 수 있는 디테일 한가득.
- **타겟 시청자**: 신기한 과학사·작전 디테일을 좋아하는 20~40대. "어떻게 그런 미친 짓을 진짜로 했지?" 류의 잡학을 즐기는 시청자.
- **서사 유형**: 영웅 서사 (니플린·부시랜드 → 퀴라소 실증 → 빗자루 박멸 → 위기 → 차세대) + 시간순. 중반 이탈 위험 낮음 (계속 새 사례 등장).
- **감정 전략**:
  - 목표 감정: 감탄·웃음 + "와 진짜?" 신기함
  - 감정 아크: 공포(키사슴) → 황당함(불임 아이디어) → 감탄(퀴라소 0%) → 폭소(주 100톤 고기·향수 일화) → 불안(위기) → 우려(내 식탁)
  - 리텐션 접근: 매 파트마다 새로운 황당 디테일 1개 (X선 몰래 빌리기 / 항공사 운송 거부 / 작업자 향수 / 뉴트리아 고기)

### CTR
- **확정 제목**: `미국이 70년 동안 매주 1억 마리의 파리를 "거세"시켜서 비행기로 뿌려온 진짜 이유`
- 썸네일 텍스트 후보:
  1. "매주 1억 마리 거세" / 미국이 70년째 하는 짓
  2. 살아있는 사슴을 파먹는 파리 / 인류가 진짜로 한 짓
  3. 비행기로 파리 뿌리는 나라 / 그게 미국임
- **이미지 컨셉**:
  - A. 상단 **비행기에서 파리떼가 쏟아지는 실사** + 하단 노란 메인카피 "매주 1억 마리 거세". photorealistic + shock. 텍스트 공간 bottom-third.
  - B. 상단 **거대한 파리 클로즈업 + 작은 비행기 실루엣** 스케일 대비. anime/hybrid + curiosity.
  - C. **사슴 클로즈업 + 충격 표정 인물**. cozy-storybook 톤(혐오 완화) + curiosity.
  - D. **방사선 마크 + 파리 + 비행기** 3요소 분산. hybrid + shock.
- 텐션 트라이앵글: 제목(정보: "거세 + 70년 + 1억"), 텍스트(감정: "거세"라는 강한 단어), 이미지(시각: 비행기 살포의 실제 장면)

### Hook & Intro
- **Hook (약 80자, ~12초)**: 비행기가 숲 위로 뭔가를 막 뿌리고 있어요. 농약 같죠? 아니에요, 이거 전부 다 파리거든요. 매주 1억 마리. 미국이 70년 동안 진심으로 하고 있는 일이에요.
- **Intro (약 220자, ~30초)**: 더 미친 건요, 이 파리들이 다 "거세된" 수컷이라는 거예요. 미국 농무부가 진짜로 매주 수억 마리의 파리를 인공적으로 불임으로 만들어서, 비행기에 실어서, 정글에 그냥 흩뿌리고 있거든요. 도대체 왜 이런 황당한 짓을 70년째 하고 있을까요? 그리고 이게 왜 지금 우리나라 소고기값이랑 직결되는 문제가 됐는지, 쉽게 말하면요, 살아있는 사슴을 파먹는 어떤 파리 한 종 때문이거든요. 한번 가볼게요.

## 패키지 B: "사슴이 살아있는 채로 파먹혔다" — 공포·혐오 훅 강화 + 미스터리 해부

### 컨셉
- **앵글**: 2016년 플로리다에서 사슴들이 살아있는 채로 파먹히기 시작한 사건에서 출발. "범인" 추적 → 정체 → 메커니즘 → 박멸 작전 → 현재 위기 순으로 미스터리 풀어가는 구조. 추리극처럼 진행.
- **핵심 약속**: "살아있는 동물을 파먹는 파리가 진짜 있다"는 호러 같은 사실에서 시작해서, 인류가 어떻게 70년에 걸쳐 이 괴물을 거의 잡았다가 다시 놓치고 있는지 그 전말을 완전히 풀어낸다.
- **타겟 시청자**: 호러·미스터리 코드를 좋아하는 20~40대. "와 이건 진짜 뭐지?" 충격을 즐기는 시청자.
- **서사 유형**: 미스터리 추적 (사건 → 범인 → 동기 → 무기 → 추격) + 폭로. 강한 훅으로 초반 이탈 차단.
- **감정 전략**:
  - 목표 감정: 공포·혐오 충격 → 안도 → 재불안
  - 감정 아크: 공포(키 사슴 두개골) → 혐오(악마의 섬 죄수) → 호기심(범인 정체) → 통쾌(박멸 성공) → 불안(재발) → 경각심(내 식탁)
  - 리텐션 접근: 초반 5분에 공포 정점 1회 박고, 중반은 "어떻게 잡았나" 통쾌함으로 내림, 후반은 "근데 다시 풀려났다"로 재상승

### CTR
- **확정 제목**: `살아있는 사슴을 머리부터 파먹는 파리 한 마리, 미국이 70년째 박멸 못 하고 있는 진짜 이유`
- 썸네일 텍스트 후보:
  1. "살아있는 사슴을 파먹는 파리" / 미국이 70년째 못 잡는 이유
  2. 두개골이 보일 때까지 / 살점이 뜯겨 나갔다
  3. 식인충 / 진짜 있다
- **이미지 컨셉**:
  - A. 상단 **사슴 + 충격 표정 + 빨간 경고 마크** (혐오 완화 처리). photorealistic + shock.
  - B. **파리 클로즈업 + "식인충" 텍스트**. anime + shock.
  - C. **숲에 비행기 + 파리떼 그림자**. hybrid + curiosity.
  - D. **사슴과 파리떼**. cozy-storybook 톤으로 충격 완화. curiosity.
- 텐션 트라이앵글: 제목(정보: "살아있는 채로 파먹는 + 70년 박멸 실패"), 텍스트(감정: "식인충"), 이미지(시각: 사슴+파리 충격)

### Hook & Intro
- **Hook (약 90자, ~13초)**: 2016년 플로리다 어느 섬에서요, 멸종위기 사슴들이 갑자기 머리를 흔들기 시작했거든요. 가서 봤더니, 정수리에 구멍이 뻥 뚫려서 두개골이 보일 정도로 살점이 뜯겨 나가 있었어요. 근데 사슴은 살아있었습니다.
- **Intro (약 200자, ~28초)**: 범인은 파리 한 종이었어요. "신세계 나선 구더기 파리"라고 불리는 녀석인데, 별명이 "식인충"이거든요. 살아있는 동물의 상처에 알을 200개씩 낳고, 12시간만에 부화한 구더기들이 나사처럼 몸을 비틀면서 산 채로 파먹어 들어가는 곤충이에요. 미국이 이걸 잡으려고 70년째 매주 1억 마리의 불임 파리를 비행기로 뿌리고 있는데요, 지금 그 작전이 무너지고 있어요. 왜 이렇게 됐는지 처음부터 설명해 드릴게요.

## 패키지 C: "내 소고기값 53% 오른 진짜 이유" — 내 삶 직격 훅 + 역추적 서사

### 컨셉
- **앵글**: 미국 소고기값이 2019년 대비 53% 오른 황당한 이유 하나가 70년 전 두 곤충학자가 시작한 X선 실험과 직결된다. "내 식탁 → 멕시코 → 파나마 공장 → 1950년대 텍사스 목장 → 1930년대 니플린의 아이디어"로 거꾸로 추적해 들어가는 역서사.
- **핵심 약속**: 마트에서 "왜 이렇게 비싸졌어?" 짜증 한 번이 70년에 걸친 인류의 곤충 박멸 작전과 직결된다는 걸 보여주는 잡학.
- **타겟 시청자**: 물가 인상에 짜증나는 30~40대 직장인. "그래서 이게 나랑 무슨 상관?" 답을 원하는 실용형 시청자.
- **서사 유형**: 역추적 (현재 → 과거) + 폭로. 첫 문장부터 "당신 일이다" 후 잡학 영역으로 끌어들이는 구조.
- **감정 전략**:
  - 목표 감정: 짜증 공감 → 호기심 → 감탄 → 우려
  - 감정 아크: 짜증·공감(소고기값) → 황당함(파리 살포) → 공포(스크루웜) → 감탄(박멸 작전) → 불안(현재 위기) → 결심(관심 갖기)
  - 리텐션 접근: 매 파트가 "그래서 내 소고기랑 무슨 상관?" 질문에 한 단계씩 답하는 구조

### CTR
- **확정 제목**: `미국 소고기 값이 53% 오른 진짜 이유 — 미국이 매주 비행기로 살포하는 1억 마리의 파리`
- 썸네일 텍스트 후보:
  1. "미국 소고기 53% 인상" / 진짜 이유는 파리 떼
  2. 한국 식탁 vs 미국 정글 / 연결고리는 파리 1마리
  3. 매주 1억 마리 / 미국이 숲에 뿌리는 이유
- **이미지 컨셉**:
  - A. **소고기 가격표 + 화살표 + 파리 클로즈업** 분할 구도. photorealistic + shock.
  - B. **마트 정육 매대 + 파리떼 그림자**. hybrid + curiosity.
  - C. **비행기 살포 + 한국 마트 합성**. anime + curiosity.
  - D. **소·파리·달러 3요소**. cozy-storybook + shock.
- 텐션 트라이앵글: 제목(정보: 53%+1억), 텍스트(감정: "내 소고기"), 이미지(시각: 마트와 정글 연결)

### Hook & Intro
- **Hook (약 75자, ~11초)**: 요즘 마트에서 미국산 소고기값 보고 깜짝 놀라셨죠? 2019년에 비해서 53%나 올랐거든요. 그 이유 중 하나가 진짜 황당한데요, 파리 한 마리 때문이에요.
- **Intro (약 210자, ~28초)**: 미국이 지금 매주 1억 마리의 파리를 비행기에 실어서 정글에 뿌리고 있어요. 그것도 70년째요. 이게 무슨 미친 짓인가 싶죠? 근데 이거 진짜 진지한 작전이거든요. 살아있는 소를 파먹는 어떤 파리 한 종을 박멸하려고 하는 건데, 이 방어선이 2022년부터 무너지기 시작하면서 멕시코까지 파리가 올라왔고, 그래서 미국 소 사육이 흔들리고, 결국 우리나라 마트 가격까지 흔들리고 있는 거예요. 처음부터 차근차근 풀어 드릴게요.

## 추천

**추천: 패키지 A (영웅 서사 + 미친 디테일)**

근거:
- 잡학위키 채널 프로필의 핵심("쓸데없이 알면 재밌는 과학 잡학")과 가장 정확히 맞물림. "주 100톤 고기, 항공사 운송 거부, 작업자 향수, X선 몰래 빌리기" 같은 디테일이 모두 술자리 안주.
- 레퍼런스 댓글 1위가 "지카 모기와 비교"인 것처럼 시청자가 원하는 것은 메커니즘 + 디테일.
- 15분 분량 소화에 가장 유리: 영웅서사 + 시간순이라 파트마다 새 인물·장소·디테일이 자연스럽게 등장.
- "거세"라는 단어가 댓글의 "심영" 밈과도 자연스럽게 매칭됨.

**B**는 호러 코드를 더 강하게 원하시면 — 다만 혐오 톤이 채널 평균보다 진할 수 있음.
**C**는 "내 삶" 직격 훅을 가장 빨리 박지만, 잡학적 재미는 A·B보다 약함.
