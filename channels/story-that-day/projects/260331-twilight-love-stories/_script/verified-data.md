- 처제: 정세아, 37세 (은이보다 10살 어린), 미술학원 강사, 청주 단독주택 혼자 거주
- 배경: 대전 지사 발령 → 청주 처제 집에서 한 달간 동거
- 서사 구조: 금지된 감정 → 처제의 20년 첫사랑 고백 → 2주 밀월 → 이별 → 명절 재회
- 영상 성과: 조회수 약 23만, 댓글 82개, 약 70분 분량

### 레퍼런스 002 — 모텔 청소부의 황혼사랑
- 주인공: 인숙, 52세, 이혼 후 모텔 객실 청소부, 딸과 단둘이 거주
- 상대: 장철수, 인테리어 회사 운영, 이혼남, 출장 단골 손님
- 배경: 모텔에서 우연한 만남 → 알몸 목격 사건을 계기로 관계 발전
- 서사 구조: 우연한 목격 → 반복 조우 → 연락처 교환 → 데이트 → 하룻밤 → 1년 후 동거
- 영상 성과: 조회수 320,978, 좋아요 3,542, 22:13 분량

### 두 레퍼런스 공통 패턴

- 핵심 감정: 외로움 → 금지된/예상치 못한 설렘 → 사랑
- 서사 장치: 인 메디아스 레스(클라이맥스 선배치) 훅, 감각 중심 묘사, 내적 갈등

### 6. 대본 활용 포인트

위 통계 데이터를 대본에 직접 삽입할 필요는 없으나, 다음과 같은 맥락에서 서사의 현실성을 뒷받침한다:

- **50대 이혼 여성의 경제적 어려움** (레퍼 002 인숙의 상황): 50대 1인가구의 30%가 이혼이 원인이며, 비정규직 취업이 현실적 선택지
