## 금액/재산 데이터

- 매달 100만 원 x 12개월 x 30년 = 총 3억 6천만 원 이상 기부 (산술적으로 검증됨)
- 재래시장 국밥집 가격: 된장국 5,000원, 소머리 국밥 7,000원 (현실적 가격대)
- 국밥집 부지 매입 제안 금액: 5억 원 ("시세보다 5배 높은 금액"이라는 설정)

## 시간/나이 데이터

- 30년 전 2살 때 헤어짐 → 현재 32세 (내부 정합성 확인됨)
- 명숙 나이 64세 (30년 전 34세 전후에 2살 아이를 둔 어머니 — 합리적)
- 아내 사별(실종) 후 7년 경과 (서사 전체에서 일관되게 유지)
- 하늘이 나이 7살 (만삭 출산 후 7년 — 일치)

## 법률/제도 데이터

- 법무부 장관은 국무위원으로 국회 인사청문회 대상 (인사청문회법에 근거)
- 검찰총장/검사 출신이 법무부 장관으로 임명된 실제 사례 다수 존재
- 살인미수 교사죄는 형법 제31조에 따라 정범과 동일하게 처벌 가능
- 사문서위조죄: 형법 제231조, 5년 이하 징역 또는 1천만 원 이하 벌금
- 살인미수 교사 + 다수 혐의 병합 시 무기징역 선고는 법리적으로 가능
- **주의**: 30년 전 상해·유기 혐의의 공소시효는 이미 완성 가능성 높음 (상해죄 7~10년, 유기죄 3~5년). 대본에서는 최근 행위(폭행, 사문서위조 등)로 처벌 근거를 설정하는 것이 현실적

## 장소/기관 데이터

- 서울 외곽에 재개발 구역 다수 존재 (현실적 설정)
- 한국심장재단 등 심장병 아동 수술비 지원 재단이 실제로 존재하며, 개인 이름으로 장기 후원 가능

## 의료 데이터

- 골절 후 적절한 치료 없이 방치하면 부정유합 등으로 영구적 보행 장애 가능 (30년간 절뚝거림 — 의학적으로 현실적)
- 장기간 영양결핍 + 열악한 환경 → 극심한 체중 감소, 면역력 저하, 호흡기 질환(각혈 포함) 가능 (의학적으로 현실적)
- 선천성 심장병은 실제 영유아에게 발생하며 수술이 필요한 경우 다수

## 서사 구조 데이터 (레퍼런스 공통 패턴)

- 두 레퍼런스 모두 동일 채널(기적같은이야기, 구독자 3만)에서 제작
- 001: 조회수 649,456 / 좋아요 18,683 / 댓글 1,200
- 002: 조회수 39,434 / 좋아요 1,890 / 영상 길이 1:43:17
- 공통 서사 공식: "오해 기반 복수 → 진실 발견 → 감정 반전 → 악인 응징 → 가족 재회"
- 공통 감정 트리거: 분노(초반) → 의심/혼란(중반) → 안타까움(중후반) → 통쾌함 + 뭉클함(후반)
- 공통 리텐션 장치: 오픈루프(결과 미리 보여주되 과정은 감춤), 디테일 묘사(신체/물건으로 감정 전달)

### 고부갈등·가족 갈등 현황

- 보건복지부 자료: 노부모 간병 주체가 2011년 "배우자-며느리-아들-딸" 순 → 2020년 "배우자-딸-아들-며느리" 순으로 변화. 10년 사이 며느리 역할이 축소되고 딸이 주도적으로 부모를 간병하는 시대로 전환 (출처: 보건복지부 2020)
- 최근 고부관계 역학 변화: "며느리 눈치 보는 시어머니"가 증가. 전통적 시어머니 구박 서사에서 역전된 양상이 나타남 (출처: 코메디닷컴 2025.10)
- 통계청: 최근 5년간 설·추석 직후 이혼 부부가 전달 대비 약 12% 증가. 고부갈등이 이혼 사유 중 하나로 지목됨 (출처: 통계청)
- 고부갈등의 근본 원인: 결혼이 여전히 '집안과 집안'의 만남으로 여겨지는 전통 유교 문화와, 개별성·자율성을 중시하는 현대 며느리 세대 간 가치관 충돌 (출처: 한국기독교상담센터)

### 노인학대 통계

- 2024년 학대피해노인: 7,167명 (전년 대비 2.0% 증가) (출처: 보건복지부 2024 노인학대 현황보고서)
- 학대 발생 장소: 가정 내 88.2%, 시설 내 8.3%
- 피해자 성별: 여성 76.6%, 남성 23.4%
- 학대 유형: 신체적 학대 43.9%, 정서적 학대 43.8%, 방임 5.6%
- 최근 4년간 노인학대 14.5% 증가 추세 (2020년 6,259건 → 2024년 7,167건) (출처: 보건복지부)
- 가해자의 65%가 배우자 또는 아들 (출처: 뉴시스 2025.06)
