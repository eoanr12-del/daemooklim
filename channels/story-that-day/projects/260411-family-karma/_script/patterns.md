## 1. 반복되는 훅 공식

### 관찰

**원문 비교**:

| 영상 | 조회수 | 훅 원문 |

| 002 | 163,690 | "와, 진짜 거짓꼴이네. 역시 내 안목은 틀리지 않았다니까. 이런 데서 평생 썩어야 어울리는 여자지. 30년을 종처럼 부려먹고도 모자라. 풀 한폭이 안자라는 죽은 땅으로 나를 내쫓은 인간들. 먼지 풀풀 날리는 정선 폐광촌 한복판에 번쩍이는 검은색 벤츠가 멈춰섰을 때 나는 직감했습니다." |

**공통 패턴 1 — 이야기 한복판 투입(In Medias Res)**:
두 영상 모두 배경 설명 없이 클라이맥스 근처 장면에서 시작한다. 001은 방파제 추락 후 구조 장면, 002는 빌런이 폐광촌에 도착하는 장면. 시청자는 맥락을 모른 채 긴장 상태에 진입한다.

**공통 패턴 2 — 빌런/가해자의 존재를 즉시 각인**:
001은 "쫓겨난 사위"의 처절한 상황으로 가해자의 존재를 암시하고, 002는 빌런 대사를 직접 인용("진짜 거짓꼴이네")하여 적대 구도를 1초 만에 확립한다. 고퍼포먼스인 002가 더 직접적이고 공격적인 빌런 대사를 사용한다.

**공통 패턴 3 — Show Don't Tell**:
001은 "살아있네. 다행이다"라는 짧은 대사로 상황의 심각성을 역설적으로 보여주고, 002는 "먼지 풀풀 날리는 정선 폐광촌 한복판에 번쩍이는 검은색 벤츠"라는 시각적 대비로 권력 구도를 보여준다. 둘 다 설명하지 않고 장면으로 보여준다.

**왜 효과적인가**:
- 맥락 없는 긴장 장면 → "이게 어떻게 된 거지?" 전방 질문 생성
- 빌런의 존재 → 즉각적 분노, 감정 투자 시작
- 구체적 감각 묘사(빗소리, 먼지, 벤츠) → 청각 콘텐츠에서 영상적 몰입 제공

**인트로 비교**:

| 영상 | 인트로 방식 |
|------|------------|
| 002 | 빌런 목적 명시("굶어 죽어가는지 구경") → 즉시 역전 예고("꿈에도 상상 못 하겠지") |
| 001 | 구독 요청 후 일상 묘사로 진입("철수의 하루가 시작됐어요. 아침 6시") |

002(고퍼포먼스)는 인트로에서도 감정 온도를 떨어뜨리지 않고 역전 약속까지 심는 반면, 001은 훅의 긴장을 일상으로 전환하며 차분하게 배경을 깐다. 두 방식 모두 유효하나, 조회수 기준으로 002의 "분노 유지 + 역전 약속" 방식이 더 강력하다.

### 설계 원칙

1. **첫 문장은 반드시 이야기의 한복판 장면에서 시작한다.** 배경 설명, 인사, 주제 소개 없이 클라이맥스 직전 또는 가장 극적인 순간의 감각적 묘사로 진입한다.
2. **훅 3문장 이내에 빌런/가해자의 악행을 구체적 대사나 행동으로 각인시킨다.** "부당한 대우를 받았다"(X) → 빌런의 직접 대사 인용 "진짜 거짓꼴이네"(O).
3. **인트로에서 역전 약속을 심는다.** "하지만 당신들은 꿈에도 상상 못 하겠지"처럼, 시청자에게 "끝까지 보면 이 악인이 무너지는 걸 본다"는 보상을 초반 30초 이내에 약속한다.
4. **설명하지 말고 보여준다.** "그는 죽을 뻔했다"(X) → "빗속에서 노란 우비를 입은 여자가 내려다보고 있었어요. 살아있네. 다행이다."(O)

## 2. 반복되는 본론 구조 & 전달 기법

### 관찰

**구조 비교**:

| 단계 | 001 (148,761) | 002 (163,690) | 공통 패턴 |
|------|--------------|--------------|----------|
| 1단계 | 억압의 일상 (처가 머슴 취급) | 학대 일상 (시집살이 30년) | **피해 축적** |
| 2단계 | 배신 발각 (불륜) + 이혼 | 내쫓김 (이혼+빈손 추방) | **최저점 추락** |
| 3단계 | 바닥 (고시원, 방파제 추락) | 죽은 땅 도착 | **절대 바닥** |
| 4단계 | 구조 + 어부 일 (현숙) | 시아버지 유산 발견 (일기장) | **반전 씨앗** |
| 5단계 | 연구 재개 + 성과 | 역이용 작전 + 증거 확보 | **역전 실행** |
| 6단계 | 전 장인에게 조건 제시 | 런칭쇼 폭로 + 체포 | **클라이맥스 응징** |
| 7단계 | 나영 몰락 + 새 출발 | 빌런 몰락 + 이름 찾기 | **빌런 말로 + 재건** |

**공통 서사 공식**: 피해 축적 → 최저점 → 반전 씨앗 → 역전 실행 → 클라이맥스 응징 → 빌런 몰락 + 주인공 재건

**전달 기법 비교**:

*스토리텔링*: 두 영상 모두 1인칭에 가까운 3인칭 서술. 001 "목구멍까지 차오른 말을 삼키면서요", 002 "심장처럼 뛰는 것 같았죠" — 내면 묘사로 이입 유도.

*대비*: 001은 "서서 일하는 철수 vs 소파의 나영", 002는 "삽을 든 노동자 vs 벤츠 앞 명품 차림 3인" — 피해자와 가해자의 물리적 위치/행동 대비가 핵심 기법.

*숫자*: 001 "14년", "45살", 002 "30년", "1000억", "징역 20년" — 두 영상 모두 피해 기간·나이·금액·처벌 수치를 구체적으로 제시. 특히 002는 "1000억"이라는 큰 숫자가 제목에도 쓰여 클릭을 견인.

*리텐션 장치 배치*:

| 위치 | 001 | 002 |
|------|-----|-----|
| 초반 | 오픈루프: "자기 자신까지도" (방파제 암시) | 오픈루프: "돌무더기 아래 무엇이 잠들어 있는지" |
| 중반 | 오픈루프: 해초→연구 연결 예고 | 오픈루프: "일기장 돌산 잊지 마라" |
| 중반 | 패턴 인터럽트: 시점 나영으로 전환 | 전환 멘트: "사냥꾼인 줄 알았던 자신들이 먹잇감" |
| 후반 | 전환 멘트: "서랍을 열었어요. 봉투가 하나" | 패턴 인터럽트: 런칭쇼 스크린 꺼짐 |

공통: 초반에 "비밀의 정체"를 열어두는 오픈루프, 중반에 시점 전환 또는 전환 멘트, 후반에 물리적 소품(서류, 스크린)을 통한 극적 전환.

### 설계 원칙

1. **7단계 서사 뼈대를 따른다**: 피해 축적(전체의 25%) → 최저점(10%) → 반전 씨앗(10%) → 역전 실행(20%) → 클라이맥스 응징(20%) → 빌런 몰락 + 재건(15%). 어떤 단계도 생략하지 않는다.
2. **피해 축적 단계에서 빌런의 악행을 직접 대사로 3회 이상 반복 제시한다.** "냅힘 말고 무릎 꿇고 제대로 닦아", "자네 같은 집안 출신이 이 자리에 앉아 있는 것만도 감사해야 해" 수준의 구체적 모욕.
3. **대비는 물리적 행동으로 보여준다.** "부당했다"(X) → "과일 썰고 술 따르는 철수 vs 소파에서 스마트폰 보는 나영"(O).
4. **숫자는 반드시 대비 쌍으로 제시한다.** "14년 머슴 → 대박", "30년 식모 → 1000억 주인" — 피해 기간 대 역전 규모의 대비.
5. **오픈루프는 초반과 중반에 각 1개씩 필수 배치한다.** 초반 오픈루프는 "이 사람이 어떻게 됐을까", 중반 오픈루프는 "이 단서가 뭘 의미하는 걸까"를 열어둔다.
6. **파트 전환 직전에 전환 멘트를 배치하여 다음 파트의 궁금증을 심는다.** "저들은 몰랐습니다. 사냥꾼인 줄 알았던 자신들이 사실은 먹잇감이라는 걸."

### 설계 원칙

1. **초반 30%는 분노 축적에 전념한다.** 빌런의 모욕 대사를 최소 3회 직접 인용하고, 주인공의 수동적 인내를 반복 묘사하여 시청자의 대리 분노 밀도를 최대치로 끌어올린다.
2. **통쾌함은 "주인공의 첫 정면 발언" 순간에 집중 폭발시킨다.** 001 "저는 그 집에서 14년을 살았어요. 사위가 아니라 머슴처럼요", 002 "당신 누구세요? 전 몰라요" — 오래 참았던 사람이 처음 고개를 드는 한 문장이 카타르시스의 정점이다.
3. **빌런의 몰락은 추상적 처벌이 아닌 생활 밀착형 디테일로 보여준다.** "벌을 받았다"(X) → "카드 정지, 홀 서빙, 흙 씹기, 면회 거절"(O). 댓글에서 "쌤통", "징벌" 반응이 구체적 몰락 장면에서 나온다.
4. **조력자 캐릭터의 무조건적 따뜻함을 충분히 서사화한다.** 현숙의 "따라와요", 시아버지의 "돌산 잊지 마라" — 아무 조건 없이 손을 내미는 인물이 있을 때 단순 복수극 이상의 감동 레이어가 생긴다.
5. **감정 흐름 설계**: 분노(초반 30%) → 긴장·동정(20%) → 희망(10%) → 통쾌함(25%) → 따뜻한 안도(15%). 분노 축적이 충분해야 통쾌함이 폭발한다.
6. **마지막 장면은 정체성 회복이나 새 출발의 따뜻한 이미지로 닫는다.** "나란히 걷는 두 사람", "제 이름은 박연이입니다" — 밤에 듣는 시청자에게 잔잔한 여운을 남긴다.

### 설계 원칙

**썸네일 텍스트 설계 원칙**:
1. 핵심 텍스트 2~3개 요소, 총 40~50자 이내. 하단 블록에 집중 배치.
2. "피해 상황 → 결과 암시/빌런 반응" 구조를 따른다.
3. 빌런의 대사 또는 감정 표현을 말풍선으로 1개 삽입하여 감정 온도를 올린다.
4. 미결형 종결("~하는데..")로 과정 호기심을 열어둔다.

**제목 설계 원칙**:
1. "[가해 행위 동사형 압축] + [역전 결과/빌런 반응]" 공식을 따른다.
2. 가해 행위는 "식모처럼", "머슴취급" 등 구체적 비유 명사를 사용한다.
3. 역전 규모를 숫자로 제시할 수 있으면 반드시 포함한다.
4. 가능하면 빌런의 반응("경악", "거품")을 포함하여 통쾌함을 예고한다.

**역할 분담 공식**:
- 썸네일 = 감정 담당 (시각적 대비 + 말풍선으로 분노/동정 유발)
- 제목 = 정보 담당 (피해 규모 + 역전 규모 + 과정 호기심)
- 겹치는 정보를 최소화하고, 썸네일과 제목이 각각 다른 자극을 준다.

**썸네일 비주얼 원칙**:
1. **A vs B 대비형 구도 필수.** 피해자(남루·고된 행색)와 가해자(부유·거만한 행색)를 좌우로 분할 배치한다.
2. 가해자 측에 부의 상징(명품, 차, 정장)을 배치하여 격차를 시각화한다.
3. 피해자는 정면 응시 또는 무표정, 가해자는 비웃음이나 거만한 표정으로 감정 대비를 만든다.
4. 하단 1/3을 어두운 반투명 배경으로 처리하여 텍스트 가독성을 확보한다.
5. 포인트 색상으로 노란색 또는 빨간색을 사용하여 시선을 잡는다.
